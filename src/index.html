About

Tally Word Counter is a free online tool to count the number of characters, words, paragraphs, and lines in your text. It can also show counts for different types of characters like letters, digits, spaces, and symbols/special characters. Make sure you have the required number of words for your essay or post by counting them instantly with Tally.


Note that this calculator was designed for use with English text. It may work for other languages, but I can't guarantee that the results will be accurate.

Tally Extension

A Tally Chrome extension is also available for even easier counting. Simply select any text, right click, and click count.

Privacy

All text processing is performed locally on your own device and is never sent to a server.


If you select a new theme or change any options, local storage will be used to store your settings locally. If you select the "Remember input text" option, your input text will also be saved between page visits. You can uncheck this option at any time to clear the saved input text. You can also wipe the saved data for this site in your browser's settings.


This site does not collect personal analytics. However, basic visitor information such as IP addresses may be passively collected by our DNS provider (Cloudflare) for security and performance purposes. This data is not used for tracking or marketing.
